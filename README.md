# SQL Daily Challenges

Hard SQL problems for Data Analysts and Data Scientists.

## Problems

1. **Day 1: Cohort Retention Analysis** - `day1-cohort-retention/`
2. **Day 2: Customer Lifetime Value** - `day2-customer-ltv/`

## How to Use

1. Go to problem folder
2. Run `schema.sql`
3. Run `data.sql`
4. Run `solution.sql`
5. Check `expected-output.md`
